How to compile the source

To compile this version of OpenICAMINUIT, it is necessary to install the CERN program library.

  • On Linux,

    OpenICAMINUIT was compiled using the Intel fortran compiler. Instructions to install this compiler on Ubuntu are available. The command to install cernlib on Ubuntu is: 

     sudo apt-get install cernlib

    A makefile is provided to compile OpenICAMINUIT. Editing this file might be required to customize the PATH of the library to your particular system. To compile the source code, use the following command 

     make

     make clean
