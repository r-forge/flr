The zip file "calk.zip" contains the following files:

readme.1st - this file.

calk_man.pdf - user's guide of the calk software in Adobe pdf format, which
includes an introduction to the method and the source code of the program.

calk - the executable file to run on Linux.

calk.exe - the executable file to run on DOS / MS Windows.

calk.c - the source code.

calktest_input.txt - an example on input file with 35 length classes, 16 age
classes, 1 data set with age data and 1 data set without age data.

calktest_output.txt - the output file resulting from running the program with the
input file "calktest_input.txt".
